Learn more about command mode
  command line mode run vimscript but we can run system binary via vimscript 
    and := is a shortcut to inspect lua values (:= vim.loop.os_uname())

    :! to run system binary like in a terminal (but there will be no aliases) (execute in working directory)

    :term open a terminal in a neovim buffer
    :norm do stuff like you are in normal modes

    :reg to show register (Current register is "". "2p to paste from register 2. "7y to yank into the 7th register)
      special register. + is the system clipboard. % is the current file name (not including the path).
        / is the last search regex. : is previous command before running the current :reg.
        alphabet are register for recording macro.
